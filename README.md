# dnrm

类似 nrm，但是速度超级无敌快 

<br />

## Usage

### install

1. 下载该项目到本地

2. 在项目根目录下执行命令

```shell
deno task install
```

### cli

```shell
# 查看当前源
dnrm

# 切换 cnpm 源
dnrm cnpm

# 切换 npm 源
dnrm npm

# 本地
dnrm npm --local

# 查看帮助
dnrm -h

# 查看版本号
dnrm -v
```
